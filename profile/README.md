# Amadeus Pro — Portable Multitrack Audio Editor

## About Amadeus Pro

Amadeus Pro provides a complete multitrack audio editing environment for users who need professional sound editing without complexity. Amadeus Pro combines multitrack recording, waveform editing, effects (reverb, delay, compressor, EQ), restoration tools (noise reduction, click removal), and spectrum analysis. Amadeus Pro enables users to record podcasts, edit interviews, restore old recordings, and master audio. Amadeus Pro runs on desktop platforms. Amadeus Pro is trusted by podcasters, voiceover artists, audio restorers, musicians, and home studio enthusiasts worldwide.

## Deep Dive — Multitrack Audio Notes

Multitrack recording: record multiple tracks simultaneously, overdub
Waveform editing: cut, copy, paste, delete, trim, fade, amplify
Effects: reverb, delay, compressor, EQ, distortion, chorus, phaser
Restoration: noise reduction, click removal, hum removal, declick
Analysis: spectrum analyzer, spectrogram, waveform statistics, frequency analysis
